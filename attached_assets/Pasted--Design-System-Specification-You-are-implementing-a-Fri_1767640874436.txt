# Design System Specification

You are implementing a "Friendly Neo-Brutalism" design system characterized by high contrast, thick outlines, bold CTAs, and playful-but-controlled energy. It must still feel professional for B2B HR. Russian UI only.

## Core Visual Language
Principles:
1) Maximum legibility and obvious actions
2) Bold hierarchy via size + outlines (not shadows)
3) Strong feedback loops (Drill Mode feels unavoidable)
4) Conversion-focused landing components
Goal: "Make the next step impossible to miss."

## Implementation Rules

### DO:
- Use thick outlines for interactive elements and drill feedback panels.
- Make primary CTA the biggest and most obvious element per screen.
- Keep lesson text short and large; progress bar bold.
- Use tags as chunky pills with borders.

### DON'T:
- Don’t add soft shadows everywhere.
- Don’t use gradients/blur.
- Don’t overuse colors: only green + semantic states + neutrals.

### Accessibility Requirements
- Contrast: >= 7:1 for body text (brutal style makes this easy)
- Focus indicators: thick + bright (ring 4px)
- Touch targets: 48x48 minimum
- prefers-reduced-motion: disable transforms (keep outline changes)

## Visual Hierarchy System
- Level 1: CTA (48px height, 3px border)
- Level 2: Progress + step title (bold)
- Level 3: Drill panels ("Почему" / "Как правильно") as big boxed sections
- Level 4: Secondary meta (mono labels: "Шаг 3/12", timestamps)
- Level 5: Disabled (opacity 0.5 + dashed border optional)

## Iconography System
- Weight: 2.5px stroke
- Corners: slightly rounded
- Grid: 24px
- Style: outlined, bold
Usage sizes as standard.

## Interaction States
Hover: translateY(-1px)
Active: translateY(1px)
Focus: 4px ring rgba(46,230,107,0.35)
Disabled: opacity 0.5; cursor not-allowed
Loading: skeleton blocks with 3px border + diagonal hatch animation (optional)

## Responsive Behavior
- Mobile-first: yes
- Scaling: stepped; keep big CTA even on mobile
- Lesson player: bottom action bar stays sticky with thick top border

## Imagery & Media
- Photography: high-contrast, human, “work in action”
- Treatment: slight warm tint to match off-white

## Micro-interactions
- Drill badge: pill "Мини-дрилл 1/2" with border + bounce 120ms (if motion allowed)
- Success: brief checkmark pop (scale 0.98→1)

## Content & Messaging (RU)
Tone: уверенно, коротко, без воды.
Rules:
- Кнопки: глагол ("Продолжить", "Проверить", "Сохранить")
- Ошибки: “что не так” + “что делать”
Examples:
- "Пользователь уже зарегистрирован — перейдите во вход."
- "Ответ не найден в базе. Отправить вопрос куратору?"
